# 1. 背景操作

```html
background-color: gold;          颜色
background-image: url("xxx");    图片
background-repeat: no-repeat;    不平铺(默认平铺)
background-repeat: repeat-x;     水平平铺
background-repeat: repeat-y;     纵向平铺
background-position: 水平 垂直;    居中(center center)
background-position-x: 100px;    平移x方向
background-position-y: 20px;     平移y方向
background-attachment: fixed;    背景固定, 不随滚动条滚动
background: url("xx") 			 是否平铺 水平偏移 垂直偏移 背景颜色(如果不设置, 默认白色, 覆盖掉之前的);
```
