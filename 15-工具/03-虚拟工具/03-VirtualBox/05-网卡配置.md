## 1.3 开机测试

配置完成后, 开启测试网络是否正常.

**注意**:

* 如果启动虚拟机后, 查看ip时, 发现某个网卡上没有分配ip, 则关机, 用其他的网卡试试

测试命令

```bash
ip a  # 查看开启的网卡中, 是否都有ip存在
ping www.baidu.com  # 测试nat是否通行
ping 宿主机192.xx的ip  # 测试连接宿主机是否通行
```
