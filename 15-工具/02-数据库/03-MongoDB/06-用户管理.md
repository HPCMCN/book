# 1. 用户管理

## 1.1 增加

```shell
use admin


db.createUser({
        user: "hpcm",
        pwd: "dong10",
        roles: ["root"]   # 创建完成后如果使用--auth开启了权限校验, 必须登陆这个用户才可以操作其他用户
    })

db.createUser({
            user: "spider",
            pwd: "dong10",
            roles: [{
                role: "readWrite",
                db: "spider"
            }]
        })
```

## 1.2 删除

```python
db.dropUser("spider")
```
