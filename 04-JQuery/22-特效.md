# 1. 特效

## 1.1 淡入淡出

```js
fadeIn()         // 淡入
fadeOut()        // 淡出
fadeToggle()     // 淡入淡出组合
```

#### 示例

```js
$("img").top().fadeToggle(2000)
```

## 1.2 卷起展开

```js
slidedown()      // 卷起
slideUp()        // 展开
slideToggle()    // 卷起展开组合
```

#### 示例

```js
$("img").top().slideToggle(2000)
```

## 1.3 淡出卷起

```js
hide()           // 淡入卷起
show()           // 淡出展开
toggle()         // 淡入卷起淡出展开组合
```
