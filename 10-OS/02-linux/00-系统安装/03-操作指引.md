# 1. man

```bash
man command:
space      下一屏
enter      下一行
b          回滚一屏    
f          前滚一屏
h          列出所有功能键    
q          退出
/word      搜索word字符串
```

# 3. 其他常用命令

```bash
which                   查找如果找到显示路径
find                    查找文件
whereis: 仅用于程序名搜索, 只搜索二进制文件
find: 在指定目录下查找, 比较浪费时间
locate: 从系统数据库中查, 效率高, 更新速度慢, 每次查询之前先使用updatedb 更新一下数据库
which: 查找环境变量中的路径中的可执行文件, 也就是说查找命令所在位置, xx/bin/xxxxx

date                 时间
gedit                打开
history              查看历史命令  !加数字调用之前用过的命令
clear                清屏 windows cls   或者ctr + l
more                 分屏查看
Htop                 视图化显示进程
>                    重定向(会覆盖)
>>                   重定向(不会覆盖)
echo                 共鸣, 相当于print
ifconfig             查询网络信息  window ipconfig
ping                 检测网路连通情况  127.0.0.1  localhost
```
