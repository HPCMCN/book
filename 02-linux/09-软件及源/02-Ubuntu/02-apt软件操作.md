# 软件操作

```bash
sudo apt-get [选项] [安装包]
```

- update: 更新源
- install: 安装程序
- remove: 拆卸程序
- install x --reinstall: 重新安装x程序
- -f install: 修复安装包
- remove x --purge: 删除安装包不保存配置
- upgrade: 升级包
- dist-upgrade: 升级系统

```bash
sudo apt-cache search x            搜索软件包
sudo apt-cache show x              获取包的相关信息
```
