# 1. 以网卡查看

## 1.1 nload安装

```bash
wget http://www.roland-riegel.de/nload/nload-0.7.4.tar.gz
tar -zxf nload-0.7.4.tar.gz
cd !$
./configure
make -j4
make install
```

## 1.2 常用命令

```bash
nload
```
