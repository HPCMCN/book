# 栈

栈/堆栈(stack), 是一种存储数据的容器. 可以存储, 访问, 删除元素. 存储操作为**先进后出**(LIFO).

![img](image/03-%E6%A0%88/%E6%A0%88.png)

# 实现

```python
class Stack(object):
    """栈"""

    def __init__(self):
        self.items = []

    def is_empty(self):
        return self.size() == 0

    def size(self):
        return len(self.items)

    def push(self, value):
        """入栈"""
        self.items.append(value)

    def pop(self):
        """出栈"""
        return self.items.pop()

    def peek(self):
        """获取栈顶元素"""
        return self.items[-1]


if __name__ == '__main__':
    stack = Stack()
    stack.push(1)
    stack.push(2)
    stack.push(3)
    print(stack.size())
    print(stack.peek())
    print(stack.pop())
    print(stack.pop())
    print(stack.pop())
    print(stack.is_empty())

```
