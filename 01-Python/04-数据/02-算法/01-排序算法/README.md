# 排序算法

Sorting algorithm(排序算法), 是一种能将一串数据按照某种特定顺序进行排列的算法

## 算法的稳定性

**稳定算法**: 排序后, 对于等值时, 保证原来的顺序

示例

```python
nums = [(1, 7), (1, 2), (2, 3)]
print(sorted(nums, key=lambda x: x[0]))
```

输出

```shell
[(1, 7), (1, 2), (2, 3)]
```

**不稳定性算法**:  排序后, 对于等值时, 不保证原来的顺序

示例

```python
nums = [(1, 7), (1, 2), (2, 3)]
print(sorted(nums))
```

输出

```shell
[(1, 2), (1, 7), (2, 3)]
```
