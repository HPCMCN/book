# 1. 界面说明

执行命令`jupyter notebook`调出浏览器, 点击新建--> python3打开notebook页面

## 1.1 页面操作类型:

* 命令模式(类似vim)
  * m:  切换到markdown模式
  * y:   切换到code模式
  * j:  向下移动
  * k:  向上移动
  * a:  向上插入
  * b: 向下插入
* 编辑模式

## 1.2 快捷键

| 快捷键        | 说明                     |
| ------------- | ------------------------ |
| `ctrl+enter`  | 执行命令(停留在当前窗口) |
| `shift+enter` | 执行命令(切换到下一行)   |
|               |                          |
