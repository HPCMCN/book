# 1. 模板

## 1.1 构建

### 1.1.1 目录层级

* cookiecutter目录结构

  ```python
  templates/
  ├── {{ cookiecutter.project_slug }}/  # 模板内容
  │   └── ...
  └── cookiecutter.json                 # 交互式参数
  ```

### 1.1.2 使用模板

* 可以将模板提交到github上, 也可以直接使用, 这里就直接调用了

  ```python
  cookiecutter --no-input mytmp
  ```
